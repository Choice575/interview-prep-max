# Changelog

## v12.12.0 (2026-07-24)
- Фильтрация, сводка прогресса и рендер экзаменационных карточек вынесены из `app.js` в тестируемый модуль `exam-ui.js`.
- Ответы, догрузка и флеш-карточки больше не используют inline JavaScript; флеш-карточки доступны с клавиатуры.
- Поиск и прогресс безопасно обрабатывают нестандартные пользовательские данные и зарезервированные ключи объектов.

## v12.11.0 (2026-07-24)
- Главная страница, mastery-карточки, история и быстрые действия вынесены из `app.js` в тестируемый модуль `home-ui.js`.
- Названия пользовательских тем и записи истории теперь выводятся через безопасные DOM-узлы без HTML-инъекций.
- Быстрые действия создаются синхронно, не дублируют обработчики и больше не используют inline JavaScript.

## v12.10.0 (2026-07-24)
- UI аналитики, готовности по грейдам и рекомендованных вопросов вынесен из `app.js` в тестируемый модуль `analytics-ui.js`.
- Карточки готовности теперь обновляются после новых ответов; подбор тем безопасно обрабатывает любые пользовательские названия.
- Кнопка «Пройти эти 10» запускает именно показанный набор рекомендаций, а не случайный микс.

## v12.9.0 (2026-07-24)
- Экспорт, schema-validation и транзакционный импорт прогресса вынесены из `app.js` в отдельный тестируемый модуль `progress-io.js`.
- Формат резервной копии, лимиты безопасности и rollback при ошибке `localStorage` сохранены без изменений.
- Новый модуль добавлен в server allowlist, app shell и offline-кеш PWA.

## v12.8.0 (2026-07-24)
- Добавлен безопасный backend-контур для внешнего AI: ключ и адрес провайдера читаются только из переменных окружения сервера.
- Персональный тренер формирует AI-разбор контрольной по агрегатам тем, точности и времени ответа; тексты вопросов, ответы и заметки не отправляются.
- При недоступном backend приложение автоматически показывает локальный разбор и продолжает работать офлайн.
- Статический сервер публикует только клиентские ассеты и не отдаёт `.env`, Git-метаданные или server-side исходники.

## v12.7.0 (2026-07-24)
- Персональный тренер показывает weekly review, сравнивает темп с прошлой неделей и корректирует объём ближайшей сессии.
- Добавлены адаптивные контрольные сессии по приоритетным темам и локальный журнал навыков с экспортом и импортом.
- PWA использует воспроизводимые PNG-иконки 192/512 и кеширует новый UI-модуль тренера для полной офлайн-работы.

## v12.6.0 (2026-07-23)
- Модальные окна получили семантику dialog, закрытие через Escape, удержание фокуса и возврат к кнопке открытия
- Таймеры экзамена, Blitz и Mock Interview считают остаток по абсолютному времени и корректно переживают фоновые вкладки
- Общий лимит Mock Interview теперь завершает интервью сразу, не перелистывая просроченные вопросы по одному

## v12.5.0 (2026-07-23)
- Импорт прогресса получил глубокую проверку, лимит размера и полный откат при ошибке `localStorage`
- Пользовательские вопросы больше не встраиваются в inline-JavaScript; поля уровня и категории экранируются
- Экзамен не рендерится на главной и выводит большие наборы вопросами по 60 карточек
